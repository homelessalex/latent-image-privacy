# Latent Image — Privacy Policy

Public hosting for the privacy policy of the **Latent Image** iOS app,
served via GitHub Pages for the App Store Privacy Policy URL.

**Live:** https://homelessalex.github.io/latent-image-privacy/

The app collects no data: no network, no analytics, no third-party SDKs.
Edit `index.html` to update the policy; GitHub Pages redeploys automatically.
